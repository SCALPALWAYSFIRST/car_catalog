# Car Catalog
## Аналитические запросы

В файле `sql/queries.sql` реализованы 5 аналитических запросов:

1. **Топ-3 самых дорогих автомобилей** - ранжирование по цене
2. **Средняя цена по марке** - анализ ценовой политики брендов  
3. **Фильтр: новые и недорогие машины** - автомобили новее 2020 года с ценой ниже 2 млн
4. **Распределение по цветам** - только популярные цвета (≥ 2 автомобиля)
5. **Классификация по возрасту** - разделение на "новые" (≤ 3 лет) и "старые" автомобили

## Структура данных

Таблица содержит 13+ записей автомобилей различных марок, годов выпуска, цветов и ценовых категорий.
