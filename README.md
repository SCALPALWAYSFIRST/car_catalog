# Car Catalog

## Аналитические запросы

В файле `sql/queries.sql` реализованы следующие аналитические запросы:

1. **Топ-5 самых дорогих автомобилей** - ранжирование по цене
2. **Статистика по маркам** - количество, средняя цена, диапазон цен
3. **Фильтр по году и цене** - автомобили 2021-2023 годов стоимостью 1-3 млн
4. **Распределение по годам выпуска** - количество моделей по годам
5. **Анализ цветов по ценовому сегменту** - классификация цвета в зависимости от средней цены
